Qual é a diferença entre linguagem de programação Server Side e Client Side?

Client Side

É executada no navegador do usuário (lado do cliente). O código é enviado para o navegador e processado ali.

Exemplos: HTML, CSS, JavaScript

Server Side

É executada no servidor. O servidor processa a requisição e envia apenas o resultado pronto para o navegador.

Exemplos: PHP, Python (com Django/Flask), Node.js, Java, C#
